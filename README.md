# ✈️ Flight Data Analysis (SQL Project)

## 📌 Objective

Analyze airline passenger data to identify trends and patterns.

## 🛠 Tools Used

* SQL
* MySQL Workbench

## 📊 Key Analysis

* Total passengers
* Year-wise growth
* Peak months
* Best performing year

## 📈 Insights

* Passenger traffic steadily increased from 1949 to 1960
* 1960 had the highest passenger count
* July and August are peak travel months
* Seasonal travel patterns observed
